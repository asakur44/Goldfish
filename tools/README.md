# Tools (Future)

This directory will hold Python tooling for progressive enhancement stages:

- `wiki-lint` — pre-commit hook for frontmatter validation, broken links, schema checks
- `wiki-state` — background script to pre-compute `wiki_state.json` (probe states, volatility labels, answer modes)
- `distill` — mining prompt for extracting outcomes from L2 conversations into shadow articles
- `metrics` — dashboard for L1 hit rate, stale read rate, cache misses

These are not needed for MVM. Add them when real pain demands it (see README.md → Month 3+ Optimizer).
